# Shopigo

**Shopigo** is a modern e-commerce platform built with **Next.js**, **Tailwind CSS**, **DaisyUI**, **Firebase Authentication**, and **MongoDB**. It allows users to browse products, add items to the cart, manage products, and securely authenticate using Firebase.

**Project Repository:** [https://github.com/Protap55/shopi-go-client](https://github.com/Protap55/shopi-go-client)
